> Official botlista api wrapper written in TypeScript

## Install

```sh
npm install botlista-wrapper-ts
```

## Usage

```js
/* Adding variable */
const Botlista = require("botlista-wrapper-ts");
/* Getting bot */
const BOT = await Botlista.GetBot(BotID: Snowflake);
/* Log bot info */
console.log(BOT)
```
